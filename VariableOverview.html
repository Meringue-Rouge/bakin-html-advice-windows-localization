Variables Used

This screen allows you to list the [Variables] specified in the game. You can also rewrite the contents of events using [Variables] from this screen.

This screen lists all the [Variables] used for events in the game. By looking at this table and following how the contents of [Variables] are changing, you can check if the flow of events is working correctly.

Clicking on a variable also opens the event sheet in which the variable is used, allowing you to immediately modify the usage of the variable in the event.

[Variables] that can store information such as [numbers], [strings], and [switches] that change within a game are very useful in creating events.

Try to take advantage of this screen!

If you are not sure what to do, please refer to the [RPG Developer Bakin Manual] from Help.
