Resources

This screen manages all resources (asset data) such as graphics and sounds that appear in the game. Resources can be added or removed, and each can be configured in detail.

RPG Developer Bakin’s resources can be divided into three main categories:

  • Terrain Stamps
    These are the so-called map chips that make up maps. Several attributes can be specified, such as “Non-Traversable”.
  • Object Stamps
    With the exception of events, almost everything placed on maps, whether 2D or 3D, are [Object Stamps]. A [Stamp] is an object that combines elements such as “model” and “texture” into a single object and then adds additional attributes to it. Basically, anything placed on maps needs to be a [Stamp].
  • Others
    These are resources that are not [Stamps], such as sounds, movies, and general-purpose data used in various parts of the game.

[Resources] is a very multifunctional screen, so there is no need to know every detail at first. When you want to make more detailed adjustments to the game, you can check again to see how each parameter is working.

If you are not sure what to do, please refer to the [RPG Developer Bakin Manual] from Help.
