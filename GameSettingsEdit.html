Game Definition

This screen specifies the basic settings and common rules for the game to be created. It allows you to specify the main cast, basic rules such as the maximum number of items that can be owned, the size of the game screen, how to control the game, and the in-game language.

First, let’s specify the game’s main cast, initial party members, etc. in the [Start Settings] section. Before doing so, it is a good idea to open the [Database] menu from the [Map Editor] and create some [Casts] (characters in this game).

Also, in the [Project Settings] section, you may want to specify the resolution at which you want to run the game you have created.

Take a peek at [Rules and Operations], [Common Terms] and [System Resources], and try out the settings that suit the game you want to create.

If you are not sure what to do, please refer to the [RPG Developer Bakin Manual] from Help.

For more in-depth information, please refer to the Bakin Wiki (in English/Japanese)
