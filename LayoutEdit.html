Layout Tool

This screen allows you to change the appearance of various screens and menus displayed in the game. You can also specify the image, position, and movement of parts to be displayed on the screen, and change the screen that opens when a button is selected on the menu.

First, from the [Screen List] on the left side of the screen, select which screen layout you would like to change to be used in which scene.

Multiple layouts can be created for each scene. During the game, you can specify which one to use by using [Layout to be Assigned] on the left side of the screen or [Layout to be Opened] in Properties.

Layouts can also be switched by events. It would be interesting to change the conversation window or the menu display according to the progress of the story.

The layouts used in games are intertwined with the game system and therefore also require complex specifications to control. While using the default layout and the layout of the sample game [Orb Stories] as a reference for your creation, try changing the layout, which can greatly affect the atmosphere of the game!

If you are not sure what to do, please refer to the [RPG Developer Bakin Manual] from Help.
