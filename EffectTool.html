Sprite Tool

This screen allows you to create animations [Sprites] that can be used in the game. 2D images can be combined and the display timing, position, angle, etc. can be specified to create a display object in motion.

[Sprites] are mainly used for 2D effects displayed in battles and events, animations of the [Casts] used in conversation scenes, and menu items that move. You can also create [Display object movements] such as fade-outs and slide-ins and use them in [Layout Tool] and so on.

Register the image you wish to display in the [Layer] section of the timeline, and specify information such as how the image will be changed. Then, by adding the next frame and specifying how the image changes from the previous frame, a [Sprite] with movement is created.

[Sprite] can be used in a variety of ways, depending on your creativity. Try it out!

If you are not sure what to do, please refer to the [RPG Developer Bakin Manual] from Help.
